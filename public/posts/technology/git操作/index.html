1. Git基础#

1.1 Git命令#

初始化git仓库：

1git init

添加文件到暂存区：

1git add 文件名

提交暂存区到仓库：

1git commit -m "提交说明"

查看仓库状态：

1git status

查看提交历史：

1git log
